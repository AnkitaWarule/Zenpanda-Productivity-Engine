# ZenPanda: AI-Driven Presence Detection & Productivity Engine 🐼

ZenPanda is a hybrid-cloud productivity ecosystem designed to facilitate deep work through real-time computer vision and system-wide interactivity. By bridging a web-based dashboard with OS-level automation, ZenPanda ensures accountability by monitoring user presence and triggering immersive celebrations upon task completion.

## 🚀 Key Features
- **AI Presence Detection:** Uses OpenCV with Haar Cascade classifiers to monitor user focus in real-time.
- **Cross-Process Communication:** A Python/Flask bridge that allows a web browser to trigger system-level desktop overlays.
- **Always-on-Top Assistant:** A transparent Tkinter-based Panda assistant that persists across all desktop applications.
- **Immersive Visuals:** High-performance CSS particle systems and web-audio synchronization for goal celebrations.

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **AI/Vision:** OpenCV
- **GUI/OS:** Tkinter
- **Frontend:** JavaScript (ES6+), Tailwind CSS, HTML5
